# StockWise-Smart-Shelf-IDP-Project

## 📌 Project Overview
StockWise is an integrated **Internet of Things (IoT) Smart Shelf Solution** built to optimize inventory control in modern retail operations. By combining hardware tracking networks with clean web interfaces, the system delivers immediate stock monitoring, predictive insights, and human-centric automation aligning with Industry 5.0 standards.

## ⚙️ Key Technical Features
- **Real-Time Data Tracking:** Integrates MFRC522 RFID readers to instantly catch stock adjustments when items leave physical shelves.
- **Automated Alerts Network:** Implements automated background scripts that push instant mail alerts to inventory clerks the second thresholds clear safe margins.
- **Dynamic Local Visualizers:** Drives localized I2C Liquid Crystal Displays to showcase up-to-date pricing and dynamic flags like "Hot-Selling!".
- **Data Engineering App:** Runs a streamlined layout using VS Code, Python, and Streamlit to map incoming serial telemetry packets seamlessly into interactive transaction datasets.

---

## 🛠️ Tech Stack & Requirements
- **Language:** Python, C/C++ (Arduino Embedded Engine)
- **Frameworks:** Streamlit Data Web Hub
- **Hardware Integration:** Arduino Uno R3, RC522 RFID Core, I2C LCD Modules, Serial Communications (`pyserial`)
